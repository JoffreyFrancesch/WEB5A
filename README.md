## Récupération du projet en local et lancement

```shell
###### get the repo ######
$ git clone https://github.com/JoffreyFrancesch/WEB5A
###### go in the right folder ######
$ cd WEB5A
###### install all npm dependencies ######
$ npm install
###### launch live development server ######
$ ng serve
```
